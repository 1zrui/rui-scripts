## 一、常用表汇总

### 住院相关
| 表名 | 说明 | 关键字段 |
|------|------|----------|
| 病人信息 | 病人主索引（67字段） | 病人ID、姓名、身份证号、住院号 |
| 病案主页 | 住院主页+出院信息（115字段） | 病人ID、主页ID、住院号、病案号、入院/出院科室ID、出院时间、状态、病案状态 |
| 住院费用记录 | 住院收费明细（71字段） | 病人ID、主页ID、实收金额、收费细目ID |
| 收费项目目录 | 收费项目（约35字段） | ID、编码、名称、类别、分类ID、执行科室 |
| 部门表 | 科室/部门 | ID、名称 |

### 病案主页核心字段
**ID字段**：病人ID、主页ID、住院号、病案号

**入院信息**：入院科室ID、入院病区ID、入院日期、入院病况、入院方式、入院病床

**出院信息**：出院科室ID、出院日期、出院病床、出院方式、住院天数

**状态**：
- 状态：0-正常住院；1-尚未入科；2-正在转科；3-已预出院
- 病案状态：1-等待审查；10-接收待审；2-拒绝审查；3-正在审查；4-审查反馈；5-审查归档；6-审查整改

**基本信息**：姓名、性别、年龄、身高、体重、职业、联系人姓名、联系人电话、联系人关系

**费用相关**：费用和（由病人费用记录触发填写）

### 收费项目目录核心字段
**ID/分类**：ID、编码、名称、类别（收费项目类别）、分类ID（药品类为NULL）

**基本信息**：规格、产地（药品/物资类）、计算单位、项目特性

**项目特性**：
- 挂号项目：0-普通；1-急诊
- 护理项目：0-一般项目；1-护理等级；2-基本护理等级

**费用相关**：
- 费用类型：甲类药、乙类药、非医保用药、A/B/C类诊疗、A/B/C类服务
- 服务对象：0-无；1-门诊；2-住院；3-门诊和住院
- 屏蔽费别：不因费别改变收费价目
- 是否变价：价格是否按固定价格

**执行科室**：0-无；1-病人所在科室；2-病区；3-操作员科室；4-指定科室；5-院外执行；6-开单人所在科室

## 二、重要字段值

### 记录状态规则（重要！）
| 统计类型 | 条件 | 说明 |
|----------|------|------|
| 人次 | `记录状态 = 1` | 有效记录 |
| 费用 | `记录状态 <> 0` | 非零金额 |

### 同时统计人次和金额
```sql
SELECT 科室,
       COUNT(DISTINCT CASE WHEN 记录状态 = 1 THEN 病人ID || 主页ID END) AS 人次,
       SUM(CASE WHEN 记录状态 <> 0 THEN 实收金额 ELSE 0 END) AS 费用
FROM 住院费用记录
WHERE ...
GROUP BY 科室
```

## 四、SQL 示例汇总

### 1. 住院药品发放明细
```sql
-- 单据 9,10 = 住院发药
-- 收费类别 5,6,7 = 西药/成药/草药
SELECT ...
FROM 药品收发记录 a, 住院费用记录 h, 病人信息 b, 药品目录 d
WHERE a.费用ID = h.ID
  AND h.病人ID = b.病人ID
  AND a.药品ID = d.药品ID
  AND h.收费类别 IN ('5','6','7')
  AND a.单据 IN (9,10)
```

### 2. 门诊药品发放明细
```sql
-- 单据 8 = 门诊发药
SELECT ...
FROM 药品收发记录 a, 门诊费用记录 h, 病人信息 b, 药品目录 d
WHERE a.费用ID = h.ID
  AND h.病人ID = b.病人ID
  AND a.药品ID = d.药品ID
  AND h.收费类别 IN ('5','6','7')
  AND a.单据 = 8
```

### 3. LIS 检验结果
```sql
SELECT t.姓名, e.检验时间, g.中文名, f.检验结果
FROM 病人医嘱记录 t
LEFT JOIN 检验标本记录 e ON t.相关ID = e.医嘱ID
LEFT JOIN 检验普通结果 f ON e.ID = f.检验标本ID
LEFT JOIN 诊治所见项目 g ON f.检验项目ID = g.ID
WHERE t.诊疗类别 = 'C'
  AND t.医嘱状态 NOT IN (-1, 4)
```

### 4. 冠心病介入诊疗明细
```sql
-- 关联：住院费用 + 病人手麻记录 + 病人医嘱记录
SELECT a.姓名, b.手术日期, b.主刀医师, c.实收金额
FROM 病人信息 a,
     病人手麻记录 b,
     住院费用记录 c
WHERE c.病人ID = a.病人ID
  AND c.病人ID = b.病人ID(+)
  AND c.主页ID = b.主页ID(+)
  AND c.医嘱序号 IN (SELECT id FROM 病人医嘱记录 WHERE 诊疗项目ID IN (SELECT id FROM 诊疗项目目录 WHERE 编码 IN ('00.8500x001','81.5400',...)))
  AND c.收据费目 = '材料费'
```

### 15. 住院科室床位使用率.txt - 床位使用率统计
- **目标**：科室床位使用率（普通床位、ICU、专科ICU）
- **核心表**：病案主页 + 部门表 + 床位状况记录_视图 + 调用存储过程
- **知识点**：
  1. **存储过程/函数调用**：
     ```sql
     from table(函数名(开始日期, 结束日期))
     ```
     - `table()` 将函数返回的结果集转为可查询的表
     - 示例：`ZL_WBY_Pak_Get病人住院天数.Zl_Wby_Fun病人住院天数`
     - 返回：病区ID、主页ID、住院天数
  
  2. **日期边界处理**：
     ```sql
     -- 入院日期边界
     case 
       when 入院日期 < 开始日期 then to_char(开始日期, 'yyyy-mm-dd')
       else to_char(入院日期, 'yyyy-mm-dd')
     end as 入院日期
     
     -- 出院日期边界（未出院用sysdate）
     case 
       when nvl(出院日期, sysdate) > 结束日期 then to_char(结束日期, 'yyyy-mm-dd')
       else to_char(出院日期, 'yyyy-mm-dd')
     end as 出院日期
     ```
  
  3. **住院天数计算**：
     ```sql
     to_date(出院日期) - to_date(入院日期) + 1
     ```
     - 出院当天算1天
  
  4. **床位使用率公式**：
     ```sql
     round(总住院日 / (现有床位数 * 时间范围天数) * 100, 1) || '%'
     ```
     - 时间范围天数 = 结束日期 - 开始日期 + 1
  
  5. **union all 合并数据**：
     ```sql
     -- 出院患者
     select 当前病区id, 主页ID, 0 from 病案主页 where 出院日期 between 开始 and 结束
     
     -- 在院患者（调用函数）
     select V_病区id, null, V_天数 from table(函数)
     ```
  
  6. **床位状况记录**：
     - 查询现有床位数
     - 排除已撤档科室：`建档时间 > 结束日期 or 撤档时间 < 开始日期`
