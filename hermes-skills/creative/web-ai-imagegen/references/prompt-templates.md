## 提示词风格指南：电影纪实风

大哥给的这个地铁提示词是一种独特的风格，跟之前的 8K 超写实摆拍完全不同。核心差异：

### 1. 视角设定（最关键的区别）
| 传统模板 | 电影纪实风 |
|---------|-----------|
| 正面/半身/全身 | **斜后方**、**乘客视角** |
| 中心构图 | 非中心、粗略构图 |
| 正对镜头 | **未察觉到镜头的存在** |
| 平视 | **镜头略向上** |

### 2. 画面瑕疵也是美学
传统模板追求通透干净，但这个风格主动加**"动态模糊"**和**"粗略的构图"**，这种"不完美"反而让画面更真实、更有故事感。

### 3. 描述顺序
```
主题句 → 年龄/美貌总述 → ①头发 → ②五官（眼/鼻/唇）→ ③脸型/脖颈/身材 → 
④服装（颜色/材质/版型）→ ⑤场景设定 → ⑥动态姿态 → ⑦拍摄视角 → ⑧画面效果
```

### 4. 情绪关键词
- 自然而然地吸引目光
- 似乎即将失去平衡
- 看起来自然放松
- 未察觉到镜头的存在

### 5. 场景 + 时间 + 人物的叠加
不只是"地铁上"，而是**"拥挤的早班列车上"** — 时间（早班）+ 状态（拥挤）+ 动作（抓扶手、失去平衡）三层叠加。

### 适用场景
- 街拍/抓拍/纪实感 → 用这组模板
- 角色设定/概念设计/三视图 → 用 8K 超写实模板

## 模板结构公式

每段提示词 = 五段信息拼接：

1. **开头指令**（可选）: "画一张图："/"Generate an image:"
2. **人物描写**: 面部特征 + 发型发色 + 表情 + 服装（颜色/材质/版型）+ 身材
3. **场景描写**: 所在环境 + 光线条件 + 背景元素 + 氛围
4. **风格参数**: 画风 + 摄影设备/参数 + 画质词（8K超高清超写实）
5. **布局要求**（三视图专用）: 左侧XXX，右侧XXX

画质词参考：`8K超高清`、`超写实`、`极致细节`、`professional portrait photography`、`shot on Sony A7IV 85mm f/1.4`、`cinematic depth of field`、`photorealistic`

## 提示词心法：两种风格体系

豆子经过今晚几轮实测，总结出两套完全不同的提示词心法：

### 体系 A：唯美摆拍风

> 用在：角色设定、宣传图、三视图、人像写真

**关键词特征**：8K超高清、超写实、完美光线、极致细节、Sony/Canon参数、纯白背景

**效果**：画面干净漂亮，但一看就是 AI 生成的「完美图片」

### 体系 B：纪实抓拍风

> 用在：街拍、日常场景、氛围感、写实主义

**关键词特征**：
- **斜后方/偷拍视角** — 不是正对镜头，而是"乘客视角"、"路人视角"
- **动态模糊** — 明确要求不清晰、有运动痕迹
- **粗略构图** — 不追求完美的三分法，而是"抓拍感"
- **下意识的动作** — "即将失去平衡的瞬间"、"没察觉到镜头"
- **不用画质词** — 别写 8K / Sony / 超写实，反而真实感更强
- **生活化细节** — 衬衫褶皱、半透明质感、拥挤车厢

**效果** 👉 像 iPhone 抓拍，真实到不像 AI 画的

### 公式对比

| | 摆拍风 A | 抓拍风 B |
|---|---|---|
| 视角 | 正面/3/4侧面，镜头平视 | 斜后方偷拍视角，镜头略仰 |
| 清晰度 | 8K超高清，发丝根根分明 | 动态模糊，带运动痕迹 |
| 构图 | 完美三分法/居中 | 粗略构图，不刻意 |
| 光线 | 暖阳/柔光/打光 | 现场自然光（车厢灯光） |
| 人物状态 | 微笑看着镜头 | 自然状态，没发现被拍 |
| 画质词 | ✅ 越多越好 | ❌ 别写，写就假 |

**结论**：想真实→用 B 系，想漂亮→用 A 系。也可以混合（比如主体写实+背景用A的精致），但核心是**不要在一张图里既要求完美无瑕又要求真实抓拍**，AI 会懵。
