# ChatGPT Web 平台参考

> ✅ 已实测（2026-06-15）

## 页面选择
- URL: `https://chatgpt.com/`
- `list_pages()` 中找标题含 "ChatGPT" 的页面
- 新开页面标题为 `"ChatGPT"`，对话开始后变为对话标题（如 `"角色设定图要求"`）

## 新建对话
- sidebar 中有 "新聊天" / "New Chat" 链接，href 指向 `https://chatgpt.com/`
- 如果当前对话内容跟要生成的图不冲突，可以直接在现有对话下发提示词
- 实测直接在旧对话里发新提示词，GPT-Image2 会正常生成图片，不会混淆

## 输入框
- textbox: `"与 ChatGPT 聊天"`（中文界面）/ `"Message ChatGPT"`（英文界面）
- uid 示例：`uid=25_31`
- 直接 `click(uid)` 聚焦后 `type_text(text, submit_key="Enter")` 即可
- **不要用 free 版的"生成图片"按钮** → 那是给 GPT-4o mini 用的简易生图，效果远不如 GPT-Image2

## 图片生成
- ✅ **实测确认：免费版也可以生图**，每天有限定张数
- GPT 自动识别生图需求 → 调用 GPT-Image2 生成，直接在对话中展示图片消息
- 不需要手动选模型，直接发带生图意图的提示词即可
- **交互提示**：GPT 出图前可能会先回复一段文字描述+布局确认，然后才开始生成图片

## 等待生成
- 生成中对话框会先出现文字响应
- 生成完成标志：图片以消息形式展示在对话中，按钮 `"已生成图片：XXX"` 出现
- 完成后的快照 DOM 结构：
  - `button "已生成图片：精致的幻想人物设定图"` — 点击展开全屏预览（uid 示例: `26_13`）
  - `button "编辑图片"`（uid 示例: `26_14`）
  - `button "分享此图片"`（uid 示例: `26_15`）
  - `button "复制回复"`（uid 示例: `26_17`）

## 下载按钮（实测流程）

> **不要试图从图片消息的缩略图直接操作** — GPT 没有 Gemini 的 "Download full size" 按钮。需先展开全屏预览。

### 步骤 A：点击图片展开全屏
```
click(uid="已生成图片按钮的uid")
```
会弹出一个 dialog lightbox：`dialog "精致的幻想人物设定图"`

### 步骤 B：全屏预览中的操作按钮
全屏预览 dialog 里的关键元素：
- `button "关闭全屏显示"` — 关闭
- `button "选择"` — 选择图片（选择模式）
- `button "宽高比"` — 调整比例
- `button "分享"` — 分享
- `button "保存"`（**关键按钮**，uid 示例: `27_14`）— 点击触发下载
- `button "展开显示"` — 更多选项

**点击 "保存" 按钮即可下载到本地**，不需要右键菜单或 hover 操作。

### 步骤 C：确认下载完成
```bash
ls -lat "$USERPROFILE/Downloads/" | head -5
# 找最新的 ChatGPT Image *.png 文件
```

## 下载文件
- 路径: `C:\\Users\\Administrator\\Downloads\\`
- 命名格式: `ChatGPT Image 2026年6月15日 00_30_56.png`（中文系统用本地化日期格式）
- 文件大小: ~1.8 MB（实测三视图 1.81 MB）
- 分辨率: 1024×1024（正方形，GPT-Image2 默认）

## GPT-Image2 实测表现
| 维度 | 表现 |
|------|------|
| 角色一致性 | ⭐⭐⭐⭐⭐ 强，完全理解"同一角色全身立绘，比例统一" |
| 布局理解 | ⭐⭐⭐⭐⭐ 精确，左侧正脸特写 + 右侧三视图排列 |
| 画质 | ⭐⭐⭐⭐ 超写实 3D 仿真，皮肤纹理保留 |
| 背景控制 | ⭐⭐⭐⭐⭐ 纯白背景无阴影，平光均匀 |
| 中文理解 | ⭐⭐⭐⭐⭐ 优秀，直接用中文提示词完全 OK |
| 生成速度 | ⭐⭐⭐ 较慢（~30-60s，且先出文字后出图） |

**结论**：三视图场景下 **GPT-Image2 >> Gemini Flash**，规范性和一致性差距明显。三视图、角色设定类首选 GPT，画面意境类可以 Gemini。

## 提示词注意事项（三视图专用）

实测有效的模板结构：

```
角色设定图，三视图（正面、侧面、背面），同一角色，全身立绘，比例统一，
纯白色背景，无阴影，平光/柔和均匀柔光打光，画面干净通透，
8K超清分辨率，超写实3D仿真画风/高清晰实摄影质感。
左侧特写正面脸部，右侧三视图。
[角色详细描述：发型发色、五官脸型、服装颜色/材质/纹路、配饰、鞋子]
柔和柔光打光，8K超清超写实3D仿真。
```

详见 `prompt-templates.md`。

## 与 Gemini 的差异总结

| 分辨率 | 1024×1024 正方形 | 1024×559 横长图 |
| 画质 | 超写实3D仿真风 | 更偏照片风 |
| 三视图规范度 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐ |
