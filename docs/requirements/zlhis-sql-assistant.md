# ZLHIS SQL 知识库 + 智能助手

## 项目概述
为医院 HIS 系统管理员/开发者提供一个本地化的 SQL 知识库工具，帮助快速查找、理解和使用 ZLHIS 数据库的 SQL 查询。

## 目标用户
- 医院信息科工作人员
- HIS 系统管理员
- 需要编写 ZLHIS SQL 的开发者

## 核心功能

### 1. SQL 知识库管理
- **自动扫描**：扫描 `SQL/` 目录下的所有 `.txt` 和 `.sql` 文件
- **分类整理**：按文件名自动分类（门诊、住院、中医、检查等）
- **内容解析**：提取 SQL 中的表名、字段名、注释

### 2. 智能搜索
- **关键词搜索**：输入关键词（如"门诊"、"中医"、"诊断"）查找相关 SQL
- **表名搜索**：搜索包含特定表的 SQL
- **字段搜索**：搜索包含特定字段的 SQL
- **模糊匹配**：支持拼音首字母、简写等模糊搜索

### 3. SQL 模板库
- **常用模式**：按科室统计、按时间筛选、关联诊断等
- **模板参数化**：支持替换科室、时间范围等参数
- **一键复制**：方便直接使用

### 4. 表关系可视化
- **核心表关系**：病案主页、病人信息、费用记录等关联图
- **字段说明**：显示字段类型、长度、业务含义
- **关联路径**：展示表之间的连接方式

### 5. SQL 生成器
- **选择表**：从表列表中选择需要的表
- **选择字段**：勾选需要查询的字段
- **生成 SQL**：自动生成基础 SELECT 语句
- **添加条件**：可视化添加 WHERE 条件

## 界面设计

### 主页
- 顶部导航栏：首页、搜索、模板、表关系、生成器
- 搜索框：居中显示，支持实时搜索
