# ZLHIS SQL 知识库 + 智能助手

## 需求概述
将 ZLHIS SQL 学习笔记整理成结构化知识库，支持智能问答。

## 功能要求

### 核心功能
1. **知识库构建**
   - 解析 `SQL/学习笔记/` 目录下的所有 .md 文件
   - 提取 SQL 片段、表结构、查询技巧
   - 建立索引（按表名、字段名、查询类型）

2. **智能问答**
   - 输入问题（如"怎么查抗菌药物"）
   - 匹配相关 SQL 片段
   - 返回答案 + SQL 示例

3. **知识更新**
   - 支持添加新的 SQL 笔记
   - 自动重建索引

### 技术要求
- Python 脚本
- 使用向量数据库（ChromaDB 或 FAISS）存储嵌入
- 使用 sentence-transformers 生成嵌入
- 命令行交互界面

### 输入参数
- `--build`：构建/重建知识库
- `--ask "问题"`：提问
- `--add file.md`：添加新知识
- `--list`：列出所有知识条目

### 输出示例
```
$ python zlhis_kb.py --ask "怎么查抗菌药物使用"

答案：
查询特殊使用级抗菌药物（抗生素=3）的出院患者用药记录：

关键表：
- 病案主页（患者信息）
- 病人医嘱记录（用药记录）
- 药品特性（抗生素级别）
- 收费项目目录（药品名称）

SQL 示例：
SELECT c.名称 科室, a.姓名, d.名称 药品名称
FROM 病案主页 a, 病人医嘱记录 b, 部门表 c, 收费项目目录 d, 药品特性 e
WHERE a.病人id = b.病人id
  AND a.出院科室id = c.id
  AND b.收费细目id = d.id
  AND b.诊疗项目id = e.药名id
  AND e.抗生素 = 3
  AND b.医嘱状态 IN (8,9)
  AND a.出院日期 > sysdate - 30

来源：ZLHIS_SQL学习笔记.md
```
