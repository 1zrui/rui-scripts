# ZLHIS数据字典查询工具 - 需求文档

## 项目概述
一个命令行工具，用于查询ZLHIS数据字典，快速查找表结构和字段说明。

## 功能需求

### 1. 查询表结构
- 输入表名（支持模糊搜索）
- 输出：表名、字段名、字段类型、字段大小、字段说明

### 2. 查询字段
- 输入字段名（支持模糊搜索）
- 输出：包含该字段的所有表

### 3. 显示格式
- 表格形式，清晰易读
- 支持分页（字段多时）

## 技术要求

### 输入
```bash
# 查询表
python dict_query.py --table "门诊费用记录"

# 查询字段
python dict_query.py --field "病人id"

# 模糊搜索
python dict_query.py --table "%费用%"
```

### 输出格式
```
表名: 门诊费用记录
字段名      字段类型    字段大小    字段说明
ID          Long        -          主键
病人ID      Long        -          病人标识
...
```

1. 能正确读取xlsx文件
