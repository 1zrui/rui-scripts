## 重要字段记录（写SQL常用）

### 1. 病人费用记录.记录状态

| 值 | 含义 |
|---|---|
| 0 | 输入尚未生效的项目 |
| 1 | 收费记录或记帐记录 |

## SQL 常用技巧

### 记录状态规则
- **人次用**：`= 1`（有效记录）
- **费用用**：`<> 0`（非零，即有效）

### DECODE 技巧
一条 SQL 同时统计人次和费用：
```sql
SELECT 
    科室,
    SUM(DECODE(记录状态, 1, 1, 0)) AS 人次,
    SUM(DECODE(记录状态, 1, 费用, 0)) AS 费用
FROM 病人费用记录
WHERE ...
GROUP BY 科室
```

## SQL 示例

### 住院费用按科室汇总
```sql
SELECT c.科室名称,
       SUM(b.实收金额) AS 费用
FROM 病人出院记录 a,
     住院费用记录 b,
     病案主页 c,
     部门表 d
WHERE a.出院时间 BETWEEN TO_DATE('2023-01-01', 'YYYY-MM-DD') AND TO_DATE('2024-03-31', 'YYYY-MM-DD')
  AND a.记录状态 = 1
  AND a.id = b.病人id
  AND a.病人id = c.病人id
  AND a.主页id = c.主页id
  AND c.出院科室id = d.id
  AND b.实收金额 IS NOT NULL
GROUP BY c.科室名称
```

### 检验结果查询（LIS）
```sql
SELECT t.病人id,
       e.检验时间,
       t.姓名,
       g.中文名 AS 检验项目,
       f.检验结果,
       g.单位,
       f.结果参考
FROM 病人医嘱记录 t,
     病案主页 a,
     病人挂号记录 b,
     病人信息 c,
     诊疗项目目录 d,
     检验标本记录 e,
     检验普通结果 f,
     诊治所见项目 g
WHERE t.病人id = a.病人id(+)
  AND t.主页id = a.主页id(+)
  AND t.挂号单 = b.no(+)
  AND t.病人id = c.病人id(+)
  AND t.诊疗项目id = d.id(+)
  AND t.相关id = e.医嘱id(+)
  AND e.id = f.检验标本id
  AND f.检验项目id = g.id
  AND t.诊疗类别 = 'C'
  AND t.医嘱状态 NOT IN (-1, 4)
  AND e.审核时间 BETWEEN TO_DATE('2022-01-01', 'YYYY-MM-DD') AND TO_DATE('2023-12-31', 'YYYY-MM-DD')
```

### 影像检查查询（PACS）
```sql
SELECT t.检查号,
       t.姓名,
       d.名称 AS 检查项目,
       t.接收日期 AS 检查时间,
       t.影像类别 AS 检查部位,
       (SELECT 名称 FROM 部门表 WHERE id = a.开嘱科室id) AS 开单科室,
       (SELECT 名称 FROM 部门表 WHERE id = a.执行科室id) AS 执行科室,
       f.设备名
FROM 影像检查记录 t,
     病人医嘱记录 a,
     病案主页 b,
     病人挂号记录 c,
     诊疗项目目录 d,
     病人信息 e,
     影像设备目录 f
WHERE t.医嘱id = a.id
  AND a.病人id = b.病人id(+)
  AND a.主页id = b.主页id(+)
  AND a.挂号单 = c.no(+)
  AND a.诊疗项目id = d.id(+)
  AND a.病人id = e.病人id(+)
  AND t.位置一 = f.设备号
  AND t.接收日期 > SYSDATE - 30
```

## 药品发放 SQL 示例

### 住院药品发放明细
```sql
SELECT b.姓名 AS 患者名,
       d.编码 AS 药品代码,
       d.名称 AS 药品名称,
       (SELECT 住院号 FROM 病案主页 WHERE 病人id = h.病人ID AND 主页id = h.主页Id) AS 病案号,
       a.审核日期 AS 发药日期,
       a.已发数量 AS 数量,
       a.零售价 AS 销售单价,
       a.零售金额 AS 收费金额,
       d.住院单位 AS 计价单位,
       (SELECT 名称 FROM 部门表 WHERE id = a.对方部门ID) AS 领药科室,
       d.规格,
       d.产地 AS 生产商,
       DECODE(h.收费类别, 5, '西药', 6, '成药', 7, '草药') AS 药品大类,
       a.no AS 处方号,
       a.批次 AS 药批次号
FROM (SELECT A.NO,
             A.药品ID,
             A.费用ID,
             A.零售价,
             MAX(a.审核日期) AS 审核日期,
             SUM(NVL(A.付数, 1) * A.实际数量) AS 已发数量,
             SUM(a.零售金额) AS 零售金额,
             A.对方部门ID,
             a.批次
      FROM 药品收发记录 A
      WHERE NVL(a.发药方式, -999) <> -1
        AND A.审核人 IS NOT NULL
        AND A.审核日期 BETWEEN TO_DATE('2022-01-01', 'YYYY-MM-DD') AND TO_DATE('2022-12-31', 'YYYY-MM-DD')
        AND A.单据 IN (9, 10)
      GROUP BY A.NO, A.药品ID, A.费用ID, A.零售价, A.对方部门ID, a.批次
      HAVING SUM(NVL(A.付数, 1) * A.实际数量) <> 0) a,
     住院费用记录 h,
     病人信息 b,
     部门表 c,
     药品目录 d
WHERE a.药品id = d.药品id
  AND h.病人ID = b.病人ID(+)
  AND a.费用ID = h.ID
  AND a.对方部门id = c.id
  AND h.收费类别 IN ('5', '6', '7')
```

### 门诊药品发放明细
```sql
-- 与住院类似，主要区别：
-- 1. 门诊费用记录 替代 住院费用记录
-- 2. a.单据 = 8 （门诊发药）
SELECT b.姓名 AS 患者名,
       d.编码 AS 药品代码,
       d.名称 AS 药品名称,
       a.no AS 处方号,
       a.审核日期 AS 发药日期,
       a.已发数量 AS 数量,
       a.零售价 AS 销售单价,
       a.零售金额 AS 收费金额,
       d.住院单位 AS 计价单位,
       (SELECT 名称 FROM 部门表 WHERE id = a.对方部门ID) AS 领药科室,
       d.规格,
       d.产地 AS 生产商,
       DECODE(h.收费类别, 5, '西药', 6, '成药', 7, '草药') AS 药品大类,
       a.批次 AS 药批次号
FROM (SELECT A.NO,
             A.药品ID,
             A.费用ID,
             A.零售价,
             MAX(a.审核日期) AS 审核日期,
             SUM(NVL(A.付数, 1) * A.实际数量) AS 已发数量,
             SUM(a.零售金额) AS 零售金额,
             A.对方部门ID,
             a.批次
      FROM 药品收发记录 A
      WHERE NVL(a.发药方式, -999) <> -1
        AND A.审核人 IS NOT NULL
        AND A.审核日期 BETWEEN TO_DATE('2022-01-01', 'YYYY-MM-DD') AND TO_DATE('2022-12-31', 'YYYY-MM-DD')
        AND A.单据 = 8
      GROUP BY A.NO, A.药品ID, A.费用ID, A.零售价, A.对方部门ID, a.批次
      HAVING SUM(NVL(A.付数, 1) * A.实际数量) <> 0) a,
     门诊费用记录 h,
     病人信息 b,
     部门表 c,
     药品目录 d
WHERE a.药品id = d.药品id
  AND h.病人ID = b.病人ID(+)
  AND a.费用ID = h.ID
  AND a.对方部门id = c.id
  AND h.收费类别 IN ('5', '6', '7')
```

## 药品发放查询要点

### 住院 vs 门诊区别
| 对比项 | 住院 | 门诊 |
|--------|------|------|
| 费用表 | 住院费用记录 | 门诊费用记录 |
| 单据号 | 9, 10 | 8 |
| 收费类别 | 5-西药,6-成药,7-草药 | 相同 |

### 核心关联
```
药品收发记录.费用ID → 门诊/住院费用记录.ID
住院费用记录.病人ID → 病人信息.病人ID
药品收发记录.药品ID → 药品目录.药品ID
药品收发记录.对方部门ID → 部门表.ID
```
