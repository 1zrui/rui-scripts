## 一、重要规则

### 记录状态规则
| 统计类型 | 条件 | 说明 |
|----------|------|------|
| 人次 | `记录状态 = 1` | 有效记录 |
| 费用 | `记录状态 <> 0` | 非零金额 |

### 同时统计人次和金额
```sql
SELECT 科室,
       COUNT(DISTINCT CASE WHEN 记录状态 = 1 THEN 病人ID || 主页ID END) AS 人次,
       SUM(CASE WHEN 记录状态 <> 0 THEN 实收金额 ELSE 0 END) AS 费用
FROM 住院费用记录
GROUP BY 科室
```

## 五、常用 SQL 示例

### 1. 住院费用按科室汇总
```sql
SELECT c.名称 AS 科室,
       COUNT(DISTINCT CASE WHEN b.记录状态 = 1 THEN b.病人ID || b.主页ID END) AS 出院人次,
       SUM(CASE WHEN b.记录状态 <> 0 THEN b.实收金额 ELSE 0 END) AS 住院收入
FROM 住院费用记录 b,
     病案主页 c,
     部门表 d
WHERE b.病人ID = c.病人ID
  AND b.主页ID = c.主页ID
  AND c.出院科室ID = d.ID
  AND c.出院日期 BETWEEN TO_DATE('2024-01-01', 'YYYY-MM-DD') AND TO_DATE('2024-12-31', 'YYYY-MM-DD')
GROUP BY c.名称
ORDER BY 住院收入 DESC
```

### 2. 门诊收入按月汇总
```sql
SELECT TO_CHAR(a.发生时间, 'YYYY-MM') AS 月份,
       b.名称 AS 科室,
       COUNT(DISTINCT CASE WHEN a.记录状态 = 1 THEN a.病人ID || a.挂号单 END) AS 门诊人次,
       SUM(CASE WHEN a.记录状态 <> 0 THEN a.实收金额 ELSE 0 END) AS 门诊收入
FROM 门诊费用记录 a,
     部门表 b
WHERE a.执行部门ID = b.ID
  AND a.发生时间 BETWEEN TO_DATE('2024-01-01', 'YYYY-MM-DD') AND TO_DATE('2024-12-31', 'YYYY-MM-DD')
GROUP BY TO_CHAR(a.发生时间, 'YYYY-MM'), b.名称
```

### 3. 门诊预约人次统计
```sql
SELECT b.名称 AS 科室,
       COUNT(DISTINCT CASE WHEN a.记录状态 = 1 THEN a.NO END) AS 预约人次,
       SUM(CASE WHEN a.记录状态 = 2 THEN 1 ELSE 0 END) AS 取消人次
FROM 病人挂号记录 a,
     部门表 b
WHERE a.执行部门ID = b.ID
  AND a.预约 = 1
  AND a.登记时间 BETWEEN TO_DATE('2025-01-01', 'YYYY-MM-DD') AND TO_DATE('2025-12-31', 'YYYY-MM-DD')
GROUP BY b.名称
```

### 4. 住院药品发放明细
```sql
SELECT b.姓名 AS 患者名,
       d.名称 AS 药品名称,
       a.审核日期 AS 发药日期,
       a.已发数量 AS 数量,
       a.零售金额 AS 收费金额,
       DECODE(h.收费类别, 5, '西药', 6, '成药', 7, '草药') AS 药品大类
FROM (SELECT A.NO, A.药品ID, A.费用ID, A.零售价,
             MAX(a.审核日期) AS 审核日期,
             SUM(NVL(A.付数, 1) * A.实际数量) AS 已发数量,
             SUM(a.零售金额) AS 零售金额,
             A.对方部门ID, a.批次
      FROM 药品收发记录 A
      WHERE NVL(a.发药方式, -999) <> -1
        AND A.审核人 IS NOT NULL
        AND A.单据 IN (9, 10)
      GROUP BY A.NO, A.药品ID, A.费用ID, A.零售价, A.对方部门ID, a.批次
      HAVING SUM(NVL(A.付数, 1) * A.实际数量) <> 0) a,
     住院费用记录 h,
     病人信息 b,
     部门表 c,
     药品目录 d
WHERE a.药品id = d.药品id
  AND h.病人ID = b.病人ID(+)
  AND a.费用ID = h.ID
  AND a.对方部门id = c.id
  AND h.收费类别 IN ('5', '6', '7')
```

### 5. LIS 检验结果查询
```sql
SELECT t.姓名,
       e.审核日期 AS 检验时间,
       g.中文名 AS 检验项目,
       f.检验结果,
       g.单位
FROM 病人医嘱记录 t
LEFT JOIN 检验标本记录 e ON t.相关ID = e.医嘱ID
LEFT JOIN 检验普通结果 f ON e.ID = f.检验标本ID
LEFT JOIN 诊治所见项目 g ON f.检验项目ID = g.ID
WHERE t.诊疗类别 = 'C'
  AND t.医嘱状态 NOT IN (-1, 4)
  AND e.审核时间 BETWEEN TO_DATE('2024-01-01', 'YYYY-MM-DD') AND TO_DATE('2024-12-31', 'YYYY-MM-DD')
```
