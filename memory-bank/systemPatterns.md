# System Patterns

## 架构模式
- 模块化设计：每个功能独立成文件/目录
- 配置与代码分离
- 文档驱动开发

## 代码规范
- 中文注释
- 清晰的函数命名
- 完整的错误处理

## 协作模式
- 布丁规划，泡泡实现
- 文件即通信
- Issue 驱动开发

## 目录结构
```
rui-scripts/
├── SQL/           # SQL 脚本
├── scripts/       # Python 工具
├── docs/          # 文档
├── memory/        # 记忆文件
└── memory-bank/   # Memory Bank 系统
```
